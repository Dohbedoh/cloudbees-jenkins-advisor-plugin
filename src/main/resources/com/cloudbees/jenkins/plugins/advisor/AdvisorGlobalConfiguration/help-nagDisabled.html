A reminder bar is displayed whenever there are no valid system or global credentials set a CloudBees user. The “Dismiss” button will hide the reminder bar for 10 minutes. Login to the CloudBees Jenkins Advisor service with your CloudBees account to begin using CloudBees Jenkins Advisor.

If this Jenkins instance does not have direct internet access, this instance can never register for the CloudBees Jenkins Advisor service. Checking this option will permanently dismiss this reminder.
